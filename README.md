Пет-проект.

Создавалась модель машинного обучения, которая училась распознавать мошеннические транзакции, делался выбор между несколькими моделями:
LogisticRegression, RandomForestClassifier, CatboostClassifier.

После неодноразового применения GridSearch для поиска гиперпараметров моделей, было определено, что CatboostClassifier имеет наивысшие показатели качества для выбранных данных.

После было выполнено обучение и достигнуты следующие показатели качества на тестовой выборке (округление до 5 знаков после запятой)):
F1_score: 0.91005
Recall: 0.83495
precision: 1.0
ROC_AUC: 0.91748
Было принято использовать эти метрики качества, так как классы не сбалансированы.

Данные взяты с https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud 
около 300к записей из которых около 0.172% мошеннические
